# MarketPulseAI Documentation

MarketPulseAI is a privacy-first browser extension for collecting and analyzing Iranian stock market data entirely on-device. The extension schedules post-close scraping, persists snapshots to IndexedDB, runs TensorFlow.js volatility analysis locally, and exports table-perfect Excel files.
